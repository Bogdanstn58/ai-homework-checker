# ai-homework-checker
# Автоматическая проверка студенческих работ с ИИ

Этот проект позволяет проверять код студентов с помощью GitHub Actions и GitHub Models (GPT-4o mini).

## Как использовать студенту
1. Склонируйте репозиторий.
2. Выполните задание в файле `solution.py`.
3. Запустите локальные тесты: `pytest tests/`.
4. Отправьте изменения в GitHub (push или pull request).
5. Получите автоматическую рецензию от ИИ в интерфейсе GitHub.

## Для преподавателя
- Файл `.github/workflows/review.yml` настраивает CI/CD.
- Секрет `GITHUB_TOKEN` должен быть добавлен в настройках репозитория.
- При каждом пуше запускаются тесты и вызов ИИ.
